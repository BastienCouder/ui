## Usage

Use `Avatar` to represent a user or an organization.

## Options

### Shape

`Avatar` can be either a circle or a square using the `shape` prop.

<ComponentPreview
  name="demos/components/core/react/data-display/avatar/shape"
  preview={`<Avatar shape="square" />
<Avatar shape="circle" />`}
/>

### Size

Use the `size` prop to control the size of the `Avatar`. The default variant is `md`.

<ComponentPreview
  name="demos/components/core/react/data-display/avatar/sizes"
  preview={`<Avatar size="sm" />
<Avatar size="md" />
<Avatar size="lg" />`}
/>
