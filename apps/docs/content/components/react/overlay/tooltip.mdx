## Options

### Variant

Use the `variant` prop to customize the tooltip style.

  name="demos/components/core/react/overlay/tooltip/variant"

</Tooltip>`}

### Placement

Use the `placement` prop to change the tooltip's position.

<ComponentPreview
  name="demos/components/core/react/overlay/tooltip/placement"

### Offset

Use the `offset` prop to adjust the distance between the tooltip and the trigger.

  name="demos/components/core/react/overlay/tooltip/offset"
