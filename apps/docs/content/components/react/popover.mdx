## Options

### Placement

Use the `placement` prop to control the side from which the popover will appear. By default, it opens from the bottom.

<ComponentPreview name="demos/ui/react/popover/placement" 
preview={`<Popover content={<Content />} placement="top"> 
  <Button variant="outline">Open popover</Button> 
</Popover>`} />

### Offset

Use the `offset` prop to adjust the distance between the popover and its trigger.

<ComponentPreview name="demos/ui/react/popover/offset" 
preview={`<Popover content={<Content />} offset={20}> 
  <Button variant="outline">Open popover</Button> 
</Popover>`} />

### Arrow

Use the `arrow` prop to add an arrow pointing to the trigger element.

<ComponentPreview name="demos/ui/react/popover/arrow" 
preview={`<Popover content={<Content />} arrow> 
  <Button variant="outline">Open popover</Button> 
</Popover>`} />
