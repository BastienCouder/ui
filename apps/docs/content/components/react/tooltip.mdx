## Options

### Variant

Use the `variant` prop to customize the tooltip style.

<ComponentPreview
  name="demos/ui/react/tooltip/variant"
  preview={`<Tooltip content="This is a default tooltip" variant="default">
    <Button>Hover</Button>
</Tooltip>`}
/>

### Placement

Use the `placement` prop to change the tooltip's position.

<ComponentPreview
  name="demos/ui/react/tooltip/placement"
  preview={`<Tooltip content="Tooltip on the right" placement="right">
    <Button>Hover me</Button>
</Tooltip>`}
/>

### Offset

Use the `offset` prop to adjust the distance between the tooltip and the trigger.

<ComponentPreview
  name="demos/ui/react/tooltip/offset"
  preview={`<Tooltip content="This tooltip is 20px away" offset={20}>
    <Button>Hover me</Button>
</Tooltip>`}
/>

### Arrow

Use the `arrow` prop to add an arrow to the tooltip.

<ComponentPreview
  name="demos/ui/react/tooltip/arrow"
  preview={`<Tooltip content="This tooltip has an arrow" arrow>
    <Button>Hover me</Button>
</Tooltip>`}
/>

### Delay

Use the `delay` prop to add a delay before the tooltip appears.

<ComponentPreview
  name="demos/ui/react/tooltip/delay"
  preview={`<Tooltip content="This tooltip appears after a delay" delay={300}>
    <Button>Hover me</Button>
</Tooltip>`}
/>
