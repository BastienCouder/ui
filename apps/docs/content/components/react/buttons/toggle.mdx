## Options

### Variants

Use the `variant` prop to control the visual style of the toggle button.
The default size is `"quiet"`.

<ComponentPreview
  name="demos/components/core/react/buttons/toggle/variants"
  preview={`<Toggle variant="quiet"><PinIcon /></Toggle>
<Toggle variant="outline"><PinIcon /></Toggle>
<Toggle variant="accent"><PinIcon /></Toggle>`}
/>

### Sizes

Use the `size` prop to control the size of the toggle button.<br/>
The default size is `"md"`.

<ComponentPreview
  name="demos/components/core/react/buttons/toggle/sizes"
  preview={`<Toggle size="sm"><PinIcon /></Toggle>
<Toggle size="md"><PinIcon /></Toggle>
<Toggle size="lg"><PinIcon /></Toggle>`}
/>

### Shapes

Use the `shape` prop to control the shape of the toggle button. The default shape is `"rectangle"`.<br/>
Icon-only toggle buttons should include an `aria-label`.

<ComponentPreview
  name="demos/components/core/react/buttons/toggle/shapes"
  preview={`<Toggle shape="square"><PinIcon /></Toggle>
<Toggle shape="circular"><PinIcon /></Toggle>
<Toggle prefix={<PinIcon />} shape="rectangle">Pin</Toggle>`}
/>

### Disabled

Use the `isDisabled` prop to disable the toggle button.

<ComponentPreview
  name="demos/components/core/react/buttons/toggle/disabled"
  preview={`<Toggle isDisabled><PinIcon /></Toggle>`}
/>

## Controlled

Use the `isSelected` and `onChange` props to control the state of the toggle button.

<ComponentPreview
  name="demos/components/core/react/buttons/toggle/controlled"

return (

  </Toggle>
